# Ai Custodi delle Ceneri

Sito ufficiale della saga sci-fi **Ai Custodi delle Ceneri** di Naq Evius.

## Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

## Sviluppo

Requisiti: Node.js e npm.

```sh
# Installa le dipendenze
npm install

# Avvia il dev server (con preview istantanea)
npm run dev

# Build di produzione
npm run build
```

Per provare il sito da un altro dispositivo sulla stessa rete:

```sh
npm run dev -- --host
```

## Deploy

Il progetto è ospitato su Vercel: il deploy di produzione avviene automaticamente al push sul branch `main`.
